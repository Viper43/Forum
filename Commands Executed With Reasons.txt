============================================= GIT HUB COMMANDS ==================================================================

1. git branch --> checks the current branch name
2. git checkout branch_name --> changes the github branch to the mentioned branch name
3. git fetch --> fetches newly created branches
4. git pull --> pulls the latest changes of the current branch
5. git stash --> to save the changes of the current branch
6. git add . --> adds all changes to be committed
7. git commit -m "message" --> commits the added changes with the provided message
8. git push --> pushes the committed changes to the branch
