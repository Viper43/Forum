========================================================= GITHUB COMMANDS ============================================================================

git branch --> checks the current branch name
git checkout branch_name --> changes the github branch to the mentioned branch name
git fetch --> fetches newly created branches
git pull --> pulls the latest changes of the current branch
git stash --> to save the changes of the current branch
git add . --> adds all changes to be committed
git commit branch_name(optional) --> commits the added changes
git push --> pushes the committed changes to the branch
